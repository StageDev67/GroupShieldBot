# GroupShieldBot 🛡️

Telegram бот для автоматической модерации групп: удаление ссылок, фильтрация мата, ограничение пересылок, система предупреждений и бан за нарушения.

## Установка

1. Клонируйте репозиторий
2. Установите зависимости: `pip install -r requirements.txt`
3. Создайте файл `.env` с токеном бота
4. Запустите: `python main.py`

## Функционал

- 🛡️ Удаление ссылок от новых пользователей (до 5 сообщений)
- 🚫 Фильтрация мата и запрещенных слов (настраиваемый список)
- 📨 Ограничение на пересылки (не более 3 подряд)
- 👥 Удаление сообщений с упоминаниями более 5 пользователей
- 👋 Кастомное приветствие новых участников
- 👋 Кастомное прощание с вышедшими участниками
- ⚠️ Система предупреждений (3 варна = бан)
- 📊 Статистика модерации
- ⚙️ Настройка правил через инлайн-кнопки
- 📋 Логирование всех действий в отдельный канал

## Команды

- `/start` - Приветственное сообщение
- `/settings` - Настройка правил группы (только для создателя)
- `/stats` - Статистика модерации
- `/warn @username [причина]` - Выдать предупреждение (админы)
- `/clear_warns @username` - Очистить предупреждения (админы)

## Требования

- Python 3.8+
- aiogram 3.x
- SQLite3
- Права администратора бота в группе
